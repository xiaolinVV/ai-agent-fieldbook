# Lab 03: Tools and Retrieval

## 目标

让 Agent 接入真实信息源。

## 要学的东西

- Function tools
- Web search
- File search
- Remote MCP
- Code interpreter

## 最小实验

做一个资料研究助手：

- 用户输入研究主题。
- Agent 使用 web search 找公开资料。
- Agent 使用 file search 查本地资料。
- 输出摘要、引用和下一步建议。

## 验收标准

- 至少使用两类工具。
- 输出中标明资料来源。
- 工具失败时有明确错误信息。
- 不把检索结果伪装成模型记忆。
